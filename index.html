Sobre

Desenvolvi um sistema web inovador capaz de identificar e classificar riscos de incêndio em imagens.


Utilizando a poderosa plataforma Google Cloud, com backend em API e banco de dados MongoDB Atlas, meu sistema analisa imagens com potenciais focos de incêndio, estimando com precisão o grau de risco com o Google Gemini.


Essa solução oferece uma ferramenta valiosa para prevenção e combate a incêndios, contribuindo para a segurança de diversos ambientes.
